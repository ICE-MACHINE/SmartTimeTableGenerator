# 🎨 Smart Timetable - Frontend

The client-side application for the Smart Timetable Generator, built with a focus on speed, clarity, and modern design.

## 🛠 Tech Stack

- **Framework:** [React 18](https://reactjs.org/) (TypeScript)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **UI Components:** [Material UI (MUI)](https://mui.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Styling:** CSS Modules & Glassmorphism UI

## 🌟 Key Modules

- **Solver Engine:** Implements the backtracking algorithm to generate conflict-free schedules.
- **Timetable Viewer:** High-performance grid for visualizing schedules with PDF/CSV export.
- **Admin Dashboard:** Management interfaces for Departments, Teachers, Rooms, and Modules.
- **Auth System:** Secure login/logout flow connected to the Node.js API.

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation
1. Install dependencies:

